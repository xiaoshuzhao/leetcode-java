### 503. 下一个更大元素 II

Next Greater Element II

#### 【单调栈】

T: O(n)

S: O(n)

```java
class Solution {
    public int[] nextGreaterElements(int[] nums) {
        int n = nums.length;
        int[] res = new int[n];
        Arrays.fill(res, -1);
        Deque<Integer> stack = new LinkedList<Integer>();
        for (int i = 0; i < n*2-1; i++){
            while (!stack.isEmpty() && nums[stack.peek()] < nums[i % n]){
               res[stack.pop()] = nums[i % n]; 
            }
            stack.push(i % n);
        }
        return res;
    }
}
```
